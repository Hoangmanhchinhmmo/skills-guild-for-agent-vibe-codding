# Bắt Đầu Với Vibe Coding

> Bạn mới làm quen với vibe coding? Hướng dẫn này sẽ giúp bạn bắt đầu xây dựng dự án chỉ trong 5 phút.

---

## Claude Code là gì?

Claude Code là một trợ lý lập trình AI chạy ngay trong terminal của bạn. Thay vì viết code từng dòng một, bạn chỉ cần **mô tả những gì bạn muốn** và Claude sẽ xây dựng cho bạn.

Hãy tưởng tượng như có một lập trình viên giàu kinh nghiệm ngồi ngay bên cạnh — chỉ khác là họ không bao giờ mệt mỏi, không bao giờ phán xét câu hỏi của bạn, và thành thạo hơn 1.282 kỹ năng chuyên biệt.

## Skills là gì?

Skills là **các khả năng chuyên biệt** giúp Claude Code trở thành chuyên gia trong từng lĩnh vực cụ thể. Khi bạn kích hoạt một skill, Claude Code sẽ nhận được hướng dẫn tập trung cho nhiệm vụ đó.

Ví dụ:
- `/react-nextjs-development` — Claude trở thành chuyên gia React/Next.js
- `/stripe-integration` — Claude trở thành chuyên gia tích hợp thanh toán
- `/seo-fundamentals` — Claude trở thành chuyên gia tư vấn SEO

## Cách Sử Dụng Skills

### Bước 1: Tìm skill phù hợp

Duyệt qua [Danh mục](../../README.vi.md#categories) hoặc sử dụng bảng [Tìm Skill](../../README.vi.md#skill-finder).

### Bước 2: Kích hoạt skill

Trong terminal Claude Code, gõ:

```
/skill-name
```

Ví dụ: `/react-nextjs-development`

### Bước 3: Mô tả những gì bạn muốn

Sau khi kích hoạt skill, chỉ cần nói cho Claude biết bạn muốn xây dựng gì:

```
/react-nextjs-development

Build me a blog with a homepage that lists all posts,
individual post pages, and a dark mode toggle.
```

Vậy là xong! Claude sẽ:
1. Tạo cấu trúc dự án
2. Viết toàn bộ code
3. Giải thích những gì đã xây dựng và lý do

## Dự Án Đầu Tiên: Tạo Landing Page

Hãy cùng xây dựng một dự án thực tế. Làm theo các bước sau:

### 1. Khởi động Claude Code

Mở terminal và chạy Claude Code.

### 2. Kích hoạt skill tạo landing page

```
/landing-page-generator
```

### 3. Mô tả trang của bạn

```
Create a landing page for my productivity app called "FocusFlow".
It should have a hero section, 3 feature cards, pricing section
with 2 tiers, and a signup form. Use a clean, modern design.
```

### 4. Thêm giao diện

```
/tailwind-patterns

Make the design responsive and add smooth scroll animations.
```

### 5. Tối ưu cho tìm kiếm

```
/seo-fundamentals

Add proper meta tags, Open Graph tags, and schema markup.
```

Chúc mừng! Bạn vừa xây dựng một landing page chuyên nghiệp chỉ với 3 skills.

## Giải Thích Các Cấp Độ Skill

| Nhãn | Ý nghĩa | Ví dụ |
|------|----------|-------|
| **[CORE]** | Thiết yếu, phù hợp với hầu hết mọi người | `react-nextjs-development` |
| **[SPEC]** | Chuyên biệt, dành cho nhu cầu cụ thể | `threejs-animation` |
| **[SEC]** | Tập trung vào bảo mật | `security-audit` |
| **[AUTO]** | Tập trung vào tự động hóa | `hubspot-automation` |

**Mới làm quen với vibe coding?** Hãy bắt đầu với các skill [CORE]. Chúng đáp ứng 80% nhu cầu của bạn.
