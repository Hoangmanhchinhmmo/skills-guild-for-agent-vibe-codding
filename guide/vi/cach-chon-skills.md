## Bảng phân biệt Skill

> Khi hai skill nghe có vẻ giống nhau, đây là cách phân biệt chúng.

| Nếu bạn đang phân vân giữa... | Hãy chọn | Khi nào |
|-------------------------------|-----------|---------|
| `react-patterns` vs `react-nextjs-development` | `react-nextjs-development` | Xây dựng ứng dụng hoàn chỉnh |
| `react-patterns` vs `react-nextjs-development` | `react-patterns` | Học các khái niệm React |
| `prompt-engineer` vs `prompt-engineering` | `prompt-engineer` | Tối ưu một prompt cụ thể |
| `prompt-engineer` vs `prompt-engineering` | `prompt-engineering` | Học các kỹ thuật viết prompt |
| `fastapi-pro` vs `fastapi-templates` | `fastapi-pro` | Xây dựng từ đầu |
| `fastapi-pro` vs `fastapi-templates` | `fastapi-templates` | Bắt đầu với template có sẵn |
| `firebase` vs `supabase-automation` | `firebase` | Hệ sinh thái Google, real-time |
| `firebase` vs `supabase-automation` | `supabase-automation` | PostgreSQL, mã nguồn mở |

| `security-audit` vs `007` | `007` | Phân tích chuyên sâu + mô hình mối đe dọa |

## Theo trình độ

### Người mới (mới bắt đầu vibe coding)

Bắt đầu với 5 skill này — chúng đáp ứng hầu hết các dự án:

1. `react-nextjs-development` — Xây dựng ứng dụng web
2. `tailwind-patterns` — Làm cho giao diện đẹp hơn
3. `supabase-automation` — Thêm cơ sở dữ liệu
4. `clerk-auth` — Thêm chức năng đăng nhập
5. `vercel-deployment` — Đưa lên mạng

### Trung bình (đã xây dựng vài dự án)

Bổ sung thêm vào bộ công cụ của bạn:

- `prompt-engineer` — Nhận kết quả tốt hơn từ AI
- `stripe-integration` — Tích hợp thanh toán
- `seo-fundamentals` — Được tìm thấy trên Google
- `nodejs-backend-patterns` — Xây dựng API
- `docker-expert` — Đóng gói ứng dụng bằng container

### Nâng cao (xây dựng hệ thống phức tạp)

Skill dành cho người dùng chuyên sâu:

- `ai-agent-development` — Xây dựng agent tự động
- `microservices-patterns` — Mở rộng kiến trúc hệ thống
- `kubernetes-architect` — Điều phối container
- `event-sourcing-architect` — Mô hình dữ liệu phức tạp
- `security-audit` — Bảo mật ứng dụng toàn diện
