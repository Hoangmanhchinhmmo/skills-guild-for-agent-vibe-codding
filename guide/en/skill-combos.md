## 1. Build a SaaS MVP (Solo Founder)

**Time estimate:** 2-4 hours | **Level:** Intermediate | **Skills:** 7

Perfect for founders who want to launch fast with auth, payments, and a polished UI.

| Step | Skill | What to do |
|------|-------|------------|
| 1 | `saas-mvp-launcher` | Plan your MVP scope, features, and architecture |
| 2 | `react-nextjs-development` | Build the frontend with Next.js App Router |
| 3 | `tailwind-patterns` | Style with Tailwind CSS, responsive design |
| 4 | `supabase-automation` | Set up database, auth, and real-time features |
| 5 | `stripe-integration` | Add subscription billing and payment processing |
| 6 | `clerk-auth` | Implement user authentication and management |
| 7 | `vercel-deployment` | Deploy to production with CI/CD |

**Why this order:** Architecture first, then frontend skeleton, then backend services, then deploy.

## Create Your Own Combo

Don't see your project type? Build your own combo:

1. **Start with architecture** — Plan before you code (`writing-plans`, `blueprint`)
2. **Pick your stack** — Choose framework skills (`react-nextjs-development`, `fastapi-pro`, etc.)
3. **Add styling** — Make it look good (`tailwind-patterns`, `frontend-design`)
4. **Connect data** — Database and API (`supabase-automation`, `firebase`, `stripe-integration`)
5. **Secure it** — Add auth and security (`clerk-auth`, `security-audit`)
6. **Optimize** — SEO, performance (`seo-fundamentals`, `web-performance-optimization`)
7. **Ship it** — Deploy (`vercel-deployment`, `docker-expert`)
