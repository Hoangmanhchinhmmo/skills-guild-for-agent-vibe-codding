# Getting Started with Vibe Coding

> New to vibe coding? This guide gets you from zero to building in 5 minutes.

---

## What is Claude Code?

Claude Code is an AI coding assistant that runs in your terminal. Instead of writing code line by line, you **describe what you want** and Claude builds it for you.

Think of it as having a senior developer sitting next to you — except it never gets tired, never judges your questions, and knows 1,282+ specialized skills.

## What are Skills?

Skills are **specialized capabilities** that make Claude Code an expert in specific areas. When you activate a skill, Claude Code gets focused instructions for that particular task.

Examples:
- `/react-nextjs-development` — Claude becomes a React/Next.js expert
- `/stripe-integration` — Claude becomes a payment integration specialist
- `/seo-fundamentals` — Claude becomes an SEO consultant

## How to Use Skills

### Step 1: Find the right skill

Browse the [Categories](../../README.md#categories) or use the [Skill Finder](../../README.md#skill-finder) table.

### Step 2: Activate it

In your Claude Code terminal, type:

```
/skill-name
```

For example: `/react-nextjs-development`

### Step 3: Describe what you want

After activating a skill, just tell Claude what to build:

```
/react-nextjs-development

Build me a blog with a homepage that lists all posts,
individual post pages, and a dark mode toggle.
```

That's it! Claude will:
1. Create the project structure
2. Write all the code
3. Explain what it built and why

## Your First Project: Build a Landing Page

Let's build something real. Follow along:

### 1. Start Claude Code

Open your terminal and run Claude Code.

### 2. Activate the landing page skill

```
/landing-page-generator
```

### 3. Describe your page

```
Create a landing page for my productivity app called "FocusFlow".
It should have a hero section, 3 feature cards, pricing section
with 2 tiers, and a signup form. Use a clean, modern design.
```

### 4. Add styling

```
/tailwind-patterns

Make the design responsive and add smooth scroll animations.
```

### 5. Optimize for search

```
/seo-fundamentals

Add proper meta tags, Open Graph tags, and schema markup.
```

Congratulations! You just built a professional landing page using 3 skills.

## Skill Levels Explained

| Tag | What it means | Example |
|-----|--------------|---------|
| **[CORE]** | Essential, works for most people | `react-nextjs-development` |
| **[SPEC]** | Specialized, for specific needs | `threejs-animation` |
| **[SEC]** | Security-focused | `security-audit` |
| **[AUTO]** | Automation-focused | `hubspot-automation` |

**New to vibe coding?** Start with [CORE] skills. They cover 80% of what you'll need.
