# How to Choose the Right Skill

> Not sure which skill to use? This guide helps you decide in 30 seconds.

---

## Decision Tree

```
What are you building?
|
+-- Web App
|   +-- Frontend only? --> react-nextjs-development + tailwind-patterns
|   +-- Full-stack? --> react-nextjs-development + nodejs-backend-patterns + supabase-automation
|   +-- Landing page? --> landing-page-generator + seo-fundamentals
|
+-- Mobile App
|   +-- Cross-platform? --> mobile-developer + react-native-architecture
|   +-- iOS only? --> ios-developer + swiftui-expert-skill
|   +-- Android only? --> android-jetpack-compose-expert
|
+-- API / Backend
|   +-- Python? --> fastapi-pro
|   +-- Node.js? --> nodejs-backend-patterns
|   +-- .NET? --> dotnet-backend
|   +-- Go? --> golang-pro
|
+-- AI / Agent
|   +-- Build an agent? --> ai-agent-development + prompt-engineer
|   +-- RAG system? --> rag-engineer + vector-database-engineer
|   +-- Use Claude API? --> claude-api + claude-code-guide
|
+-- E-commerce
|   +-- Shopify? --> shopify-development
|   +-- Custom? --> stripe-integration + react-nextjs-development
|
+-- Not building, need help
    +-- Fix bugs? --> debugger + bug-hunter
    +-- Improve code? --> code-reviewer + clean-code
    +-- Security check? --> security-audit
    +-- Deploy? --> vercel-deployment or docker-expert
```

## Disambiguation Table

> When two skills sound similar, here's how they differ.

| If you're choosing between... | Use this one | When |
|-------------------------------|-------------|------|
| `react-patterns` vs `react-nextjs-development` | `react-nextjs-development` | Building a full app |
| `react-patterns` vs `react-nextjs-development` | `react-patterns` | Learning React concepts |
| `prompt-engineer` vs `prompt-engineering` | `prompt-engineer` | Optimizing a specific prompt |

## By Experience Level

### Beginner (just starting with vibe coding)

Start with these 5 skills — they cover most projects:

1. `react-nextjs-development` — Build web apps
2. `tailwind-patterns` — Make them look good
3. `supabase-automation` — Add a database
4. `clerk-auth` — Add login
5. `vercel-deployment` — Put it online

### Intermediate (built a few things)

Add these to your toolkit:

- `prompt-engineer` — Get better results from AI
- `stripe-integration` — Accept payments
- `seo-fundamentals` — Get found on Google
- `nodejs-backend-patterns` — Build APIs
- `docker-expert` — Containerize your apps

### Advanced (building complex systems)

Power user skills:

- `ai-agent-development` — Build autonomous agents
- `microservices-patterns` — Scale your architecture
- `kubernetes-architect` — Orchestrate containers
- `event-sourcing-architect` — Complex data patterns
- `security-audit` — Lock down your apps
