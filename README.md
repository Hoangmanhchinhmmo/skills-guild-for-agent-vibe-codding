## What is Vibe Coding with Claude Code?

**Vibe coding** is building software with AI — even if you're not a traditional developer. Instead of writing every line of code yourself, you describe what you want and an AI coding assistant like **Claude Code** builds it for you.

Claude Code has **1,282+ specialized skills** that turn it into an expert for any task: from building React apps to setting up payment systems, from writing SEO-optimized content to deploying to production.

**This repo is the complete guide to every Claude Code skill.** We organize all skills into **21 categories**, explain **what each one does** in plain English, and show you **which skills to combine** for real projects — so you can build faster, even without deep technical knowledge.

Whether you're a founder building a SaaS MVP, a designer who wants to code, a marketer automating workflows, or a developer exploring AI-assisted development — start here.

## Quick Start: How to Use Claude Code Skills

Using Claude Code skills takes 3 steps. No setup, no install, no config required.

```
1. Browse the 21 categories below (or use the Skill Finder table)
2. Pick a skill — or pick a combo for full project workflows
3. Type /skill-name in Claude Code and describe what you want to build
```

**Example:** Type `/react-nextjs-development` then say "Build me a blog with dark mode and a homepage that lists all posts." Claude Code handles the rest.

## Frequently Asked Questions

<details>
<summary><strong>What is vibe coding?</strong></summary>

Vibe coding is a term for building software using AI assistants like Claude Code. Instead of writing every line of code manually, you describe what you want in plain English and the AI generates the code. It makes software development accessible to founders, designers, marketers, and anyone with an idea — even without a traditional programming background.
</details>

<details>
<summary><strong>What is Claude Code?</strong></summary>

Claude Code is Anthropic's AI coding assistant that runs in your terminal. It can write code, debug issues, deploy projects, and handle complex development tasks. Skills are specialized modes that make Claude Code an expert in specific areas like React development, database design, or SEO optimization.
</details>

<details>
<summary><strong>Do I need programming experience to use these skills?</strong></summary>

No. Many skills are beginner-friendly and designed for people who are new to coding. Start with [CORE] skills and follow our <a href="guide/en/getting-started.md">Getting Started guide</a>. The Skill Finder table above matches your goals to the right skills regardless of experience level.
</details>

<details>
<summary><strong>How do I install a Claude Code skill?</strong></summary>

You don't need to install anything. Skills are built into Claude Code. Just type <code>/skill-name</code> in your Claude Code terminal (e.g., <code>/react-nextjs-development</code>) and start describing what you want to build.
</details>

<details>
<summary><strong>Can I use multiple skills together?</strong></summary>

Yes! That's what our <a href="guide/en/skill-combos.md">Skill Combos</a> are for. Combining skills lets you build complete projects. For example, combine <code>saas-mvp-launcher</code> + <code>stripe-integration</code> + <code>clerk-auth</code> to build a full SaaS product with payments and authentication.
</details>

<details>
<summary><strong>How is this different from other awesome lists?</strong></summary>

Most awesome lists just link to tools. This repo is an actionable skill catalog: every skill has a plain-English description, a difficulty level, and — for top skills — example prompts, recommended combinations, and guidance on when to use alternatives. The Skill Combos feature provides step-by-step project recipes you won't find anywhere else.
</details>
