## Enriched Skills

### `backend-architect`
**What you need:** Helps you design the big picture of your backend â€” how your services talk to each other, where your data lives, and how everything scales as your app grows. Think of it as the blueprint phase before you start building.
**Level:** Advanced
**Tags:** #architecture #backend #scalability #system-design #microservices #database
**When to use:**
- You are starting a new project and need to decide on the overall backend structure
- You need to figure out how to split your app into services, APIs, and databases
- Your app is growing and you need to rethink how things are organized to handle more users
**Combine with:** `brainstorming` > `backend-architect` > `fastapi-pro` > `auth-implementation-patterns`
**Example prompt:** > /backend-architect Design the backend architecture for a marketplace app with user profiles, listings, real-time chat, and payment processing
**Don't use when:** You already have your architecture and just need to write code â€” use `fastapi-pro` or `nodejs-backend-patterns` instead

### `fastapi-pro`
**What you need:** Builds fast, modern Python APIs using FastAPI. It handles async requests (multiple things at once), automatic API documentation, data validation, and all the production details so your API is fast and reliable.
**Level:** Intermediate
**Tags:** #python #fastapi #api #async #rest #backend #pydantic
**When to use:**
- You are building a Python backend API from scratch or adding new endpoints
- You need high-performance async endpoints that handle many requests at once
- You want automatic API docs and strong data validation built in
**Combine with:** `backend-architect` > `fastapi-pro` > `auth-implementation-patterns` > `docker-compose`
**Example prompt:** > /fastapi-pro Build a REST API for a todo app with user accounts, CRUD operations, pagination, and SQLAlchemy database models
**Don't use when:** You prefer JavaScript/TypeScript for your backend â€” use `nodejs-backend-patterns` or `hono` instead

### `nodejs-backend-patterns`
**What you need:** Gives you production-ready patterns for building Node.js backends with Express or Fastify. Covers routing, middleware, error handling, database connections, and all the things that make a backend solid and maintainable.
**Level:** Intermediate
**Tags:** #nodejs #express #fastify #javascript #typescript #api #backend #middleware
**When to use:**
- You are building a JavaScript or TypeScript backend with Express or Fastify
- You need reliable patterns for error handling, logging, and request validation
- You want to structure your Node.js project so it stays clean as it grows
**Combine with:** `backend-architect` > `nodejs-backend-patterns` > `auth-implementation-patterns` > `docker-compose`
**Example prompt:** > /nodejs-backend-patterns Build an Express API with TypeScript, Zod validation, structured error handling, and PostgreSQL using Prisma
**Don't use when:** You want a Python backend â€” use `fastapi-pro` or `django-pro` instead
