## Enriched Skills

### `claude-code-guide`
**What you need:** A complete walkthrough for getting the most out of Claude Code, the command-line tool that lets you use Claude directly in your terminal to write code, manage files, and build projects.
**Level:** Beginner
**Tags:** #claude-code #cli #productivity #developer-tools #workflow
**When to use:**
- You are new to Claude Code and want to learn how to use it effectively
- You want to discover shortcuts, best practices, and power-user tips for the Claude Code CLI
- You need help setting up Claude Code for a specific project or workflow
**Combine with:** `claude-api` > `claude-code-guide` > `prompt-engineer` > `context-fundamentals`
**Example prompt:** > /claude-code-guide Show me how to set up Claude Code for a Next.js project with custom CLAUDE.md instructions and efficient context management
**Don't use when:** You want to build apps that call the Claude API programmatically -- use `claude-api` instead

### `claude-api`
**What you need:** Shows you how to connect your own applications to Claude's brain by using the Anthropic API and SDK, so your app can send questions and get intelligent responses back.
**Level:** Intermediate
**Tags:** #claude #api #anthropic #sdk #ai-integration #llm
**When to use:**
- You want to add AI-powered features to your web app, mobile app, or backend service
- You need to call Claude from your code to generate text, answer questions, or process data
- You are building a chatbot, content generator, or any tool that needs Claude's capabilities
**Combine with:** `claude-code-guide` > `claude-api` > `prompt-engineer` > `llm-structured-output`
**Example prompt:** > /claude-api Build a Node.js service that uses the Claude API to summarize customer support tickets and categorize them by urgency
**Don't use when:** You just want to use Claude interactively in your terminal -- use `claude-code-guide` instead
