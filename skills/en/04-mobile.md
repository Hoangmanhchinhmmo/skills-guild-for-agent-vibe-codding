## Enriched Skills

### `mobile-developer`
**What you need:** Helps you build mobile apps that work on both iPhone and Android using popular frameworks like React Native, Flutter, or native tools -- from setting up your project to shipping it to users.
**Level:** Intermediate
**Tags:** #mobile #react-native #flutter #ios #android #cross-platform
**When to use:**
- You want to build one app that runs on both iOS and Android
- You are choosing between React Native, Flutter, or going fully native and need guidance
- You need to set up a mobile project with navigation, API calls, and device features like camera or push notifications
**Combine with:** `mobile-design` > `mobile-developer` > `expo-deployment` > `app-store-optimization`
**Example prompt:** > /mobile-developer Build a cross-platform fitness tracker app with React Native that tracks daily steps and shows weekly progress charts
**Don't use when:** You only need to build for one platform -- use `ios-developer` for Swift/SwiftUI or `android-jetpack-compose-expert` for Android-only apps
