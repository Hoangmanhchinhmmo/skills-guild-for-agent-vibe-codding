## Enriched Skills

### `supabase-automation`
**What you need:** Automates tasks in Supabase, the open-source Firebase alternative, including setting up database tables, writing row-level security policies, managing user authentication, and triggering actions based on data changes.
**Level:** Intermediate
**Tags:** #supabase #automation #database #auth #postgres #row-level-security
**When to use:**
- You want to automate the creation of Supabase tables, policies, and edge functions
- You need to set up Supabase auth flows and row-level security without doing it all manually
- You are building a project on Supabase and want to script your infrastructure setup
**Combine with:** `supabase-automation` > `nextjs-supabase-auth` > `postgresql` > `stripe-integration`
**Example prompt:** > /supabase-automation Set up a Supabase project with a users table, a posts table with row-level security, email/password auth, and an edge function that sends a welcome email on signup
**Don't use when:** You need a NoSQL or document database -- use `firebase` instead, or use `database` for general database work
