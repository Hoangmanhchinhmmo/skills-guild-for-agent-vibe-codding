## Enriched Skills

### `react-nextjs-development`
**What you need:** Your go-to skill for building modern web apps with React and Next.js (version 14 and newer). It knows about Server Components, the App Router, data fetching, and all the latest patterns so you do not have to.
**Level:** Intermediate
**Tags:** #react #nextjs #frontend #web-app #server-components #app-router
**When to use:**
- You are building a new web application from scratch with React and Next.js
- You need to set up pages, layouts, API routes, or data loading in a Next.js project
- You want to use modern features like Server Components, streaming, or server actions
**Combine with:** `writing-plans` > `react-nextjs-development` > `tailwind-patterns` > `auth-implementation-patterns`
**Example prompt:** > /react-nextjs-development Build me a dashboard page with a sidebar, top nav, and a data table that loads user data from an API route
**Don't use when:** You are building a simple static site with no interactivity â€” consider `astro` or `landing-page-generator` instead

### `tailwind-patterns`
**What you need:** Helps you style your app quickly using Tailwind CSS version 4. Instead of writing custom CSS, you use short class names directly in your HTML to control colors, spacing, layout, and responsiveness.
**Level:** Beginner
**Tags:** #css #tailwind #styling #responsive #design-system #ui
**When to use:**
- You want to style components without writing separate CSS files
- You need responsive designs that look good on phones, tablets, and desktops
- You are building a consistent look across your app with reusable style patterns
**Combine with:** `react-nextjs-development` > `tailwind-patterns` > `shadcn` > `frontend-design`
**Example prompt:** > /tailwind-patterns Style a pricing card with a highlighted "popular" tier, hover effects, and a mobile-friendly layout
**Don't use when:** Your project uses a completely different CSS approach like CSS Modules or styled-components â€” use `frontend-design` for general styling guidance
