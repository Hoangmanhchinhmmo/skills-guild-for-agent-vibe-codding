## Enriched Skills

### `ai-agent-development`
**What you need:** Helps you build AI agents — programs that can think, make decisions, and take actions on their own. It covers how to give agents tools, memory, and the ability to break big tasks into smaller steps.
**Level:** Advanced
**Tags:** #ai-agent #autonomous #tools #memory #decision-making #architecture
**When to use:**
- You want to build an AI that can do tasks by itself, like researching topics or managing workflows
- You need an agent that can use tools (search the web, run code, call APIs) to get things done
- You are designing a system where multiple AI agents work together
**Combine with:** `prompt-engineer` > `ai-agent-development` > `agent-memory-systems` > `agent-evaluation`
**Example prompt:** > /ai-agent-development Build me an AI agent that monitors my GitHub repos, summarizes new issues daily, and drafts response suggestions
**Don't use when:** You just need a simple chatbot or one-shot AI response — use `prompt-engineer` or `claude-api` instead

### `prompt-engineer`
**What you need:** Teaches you how to write better instructions for AI models so they give you exactly what you want. Small changes in how you ask can make a huge difference in the quality of results.
**Level:** Beginner
**Tags:** #prompts #ai #optimization #instructions #few-shot #chain-of-thought
**When to use:**
- Your AI responses are not quite right and you want to improve them
- You are building an app that sends prompts to an AI and need reliable, consistent output
- You want to learn techniques like giving examples, step-by-step thinking, or role-based prompts
**Combine with:** `prompt-engineer` > `ai-agent-development` > `llm-structured-output`
**Example prompt:** > /prompt-engineer Help me write a system prompt for a customer support bot that stays friendly, answers from our FAQ, and knows when to escalate to a human
**Don't use when:** You are looking for a library of ready-made prompts — use `prompt-library` instead

### `deep-research`
**What you need:** Conducts thorough research across multiple sources automatically. It searches, reads, cross-references, and synthesizes information step by step — like having a research assistant that does not stop at the first result.
**Level:** Intermediate
**Tags:** #research #multi-step #analysis #synthesis #web-search #autonomous
**When to use:**
- You need a comprehensive answer that requires checking multiple sources
- You want to understand a complex topic with pros, cons, and tradeoffs
- You need up-to-date information that goes beyond what the AI already knows
**Combine with:** `deep-research` > `writing-plans` > `executing-plans`
**Example prompt:** > /deep-research Compare the top 5 headless CMS options for a Next.js blog — pricing, features, developer experience, and which is best for a solo developer
**Don't use when:** You need a quick factual answer from a single source — use `search-specialist` for fast lookups

### `search-specialist`
**What you need:** Finds specific information on the web quickly and accurately. It knows how to craft the right search queries, pick the best sources, and pull out exactly what you need without the noise.
**Level:** Beginner
**Tags:** #search #web #information-retrieval #lookup #fact-checking
**When to use:**
- You need to find a specific piece of information like documentation, a code example, or a recent announcement
- You want to check if something is true or find the latest version of a library
- You need current information that the AI might not have in its training data
**Combine with:** `search-specialist` > `deep-research` > `brainstorming`
**Example prompt:** > /search-specialist Find the official migration guide for upgrading from Next.js 14 to Next.js 15 and summarize the breaking changes
**Don't use when:** You need deep, multi-source analysis on a broad topic — use `deep-research` for that
