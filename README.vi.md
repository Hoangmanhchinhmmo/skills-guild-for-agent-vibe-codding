## Vibe Coding với Claude Code là gì?

**Vibe coding** là xây dựng phần mềm bằng AI — ngay cả khi bạn không phải developer truyền thống. Thay vì tự viết từng dòng code, bạn mô tả thứ mình muốn và trợ lý AI như **Claude Code** sẽ xây dựng cho bạn.

Claude Code có **1,282+ skills chuyên biệt** biến nó thành chuyên gia cho bất kỳ nhiệm vụ nào: từ xây ứng dụng React đến thiết lập hệ thống thanh toán, từ viết nội dung SEO đến triển khai production.

**Repo này là hướng dẫn đầy đủ cho mọi Claude Code skill.** Chúng tôi tổ chức tất cả skills vào **21 danh mục**, giải thích **mỗi skill làm gì** bằng tiếng Việt dễ hiểu, và chỉ cho bạn **kết hợp skills nào** cho dự án thực tế — để bạn xây dựng nhanh hơn, ngay cả khi chưa có kiến thức kỹ thuật sâu.

Dù bạn là founder đang xây MVP, designer muốn code, marketer tự động hóa quy trình, hay developer khám phá AI — hãy bắt đầu từ đây.

## Bắt đầu nhanh: Cách sử dụng Claude Code Skills

Sử dụng Claude Code skills chỉ cần 3 bước. Không cần cài đặt, không cần cấu hình.

```
1. Duyệt 21 danh mục bên dưới (hoặc dùng Bảng tìm Skill)
2. Chọn một skill — hoặc chọn combo cho dự án đầy đủ
3. Gõ /tên-skill trong Claude Code và mô tả thứ bạn muốn xây
```

**Ví dụ:** Gõ `/react-nextjs-development` rồi nói "Xây cho tôi một blog có dark mode và trang chủ hiển thị danh sách bài viết." Claude Code lo phần còn lại.

## Hiểu về mức độ Skill

Mỗi Claude Code skill có tag phân loại để giúp bạn quyết định sử dụng:

| Tag | Ý nghĩa | Phù hợp với |
|-----|---------|-------------|
| **[CORE]** | Skill thiết yếu, được dùng bởi hầu hết dự án | Tất cả — bắt đầu từ đây |
| **[SPEC]** | Chuyên biệt cho framework hoặc trường hợp cụ thể | Khi dự án cần |
| **[SEC]** | Tập trung kiểm tra và tăng cường bảo mật | Người quan tâm bảo mật |
| **[AUTO]** | Tự động hóa và tích hợp quy trình | Người tự động hóa workflow |

**Mới bắt đầu vibe coding?** Hãy tập trung vào skills [CORE] trước. Chúng đáp ứng 80% nhu cầu của hầu hết dự án.

## Câu hỏi thường gặp

<details>
<summary><strong>Vibe coding là gì?</strong></summary>

Vibe coding là thuật ngữ chỉ việc xây dựng phần mềm sử dụng trợ lý AI như Claude Code. Thay vì viết từng dòng code thủ công, bạn mô tả thứ mình muốn bằng ngôn ngữ tự nhiên và AI tạo code cho bạn. Nó giúp việc phát triển phần mềm trở nên dễ tiếp cận với founders, designers, marketers, và bất kỳ ai có ý tưởng — ngay cả khi chưa có nền tảng lập trình truyền thống.
</details>

<details>
<summary><strong>Claude Code là gì?</strong></summary>

Claude Code là trợ lý lập trình AI của Anthropic chạy trong terminal. Nó có thể viết code, sửa lỗi, triển khai dự án, và xử lý các tác vụ phát triển phức tạp. Skills là các chế độ chuyên biệt biến Claude Code thành chuyên gia trong các lĩnh vực cụ thể như phát triển React, thiết kế database, hay tối ưu SEO.
</details>

<details>
<summary><strong>Tôi có cần kinh nghiệm lập trình không?</strong></summary>

Không. Nhiều skills thân thiện với người mới và được thiết kế cho người chưa từng code. Bắt đầu với skills [CORE] và làm theo <a href="guide/vi/bat-dau.md">Hướng dẫn Bắt đầu</a>. Bảng tìm Skill ở trên ghép mục tiêu của bạn với skills phù hợp bất kể trình độ.
</details>

<details>
<summary><strong>Làm sao để cài đặt Claude Code skill?</strong></summary>

Bạn không cần cài đặt gì cả. Skills được tích hợp sẵn trong Claude Code. Chỉ cần gõ <code>/tên-skill</code> trong terminal Claude Code (ví dụ: <code>/react-nextjs-development</code>) và bắt đầu mô tả thứ bạn muốn xây.
</details>

<details>
<summary><strong>Có thể dùng nhiều skills cùng lúc không?</strong></summary>

Có! Đó là mục đích của <a href="guide/vi/ket-hop-skills.md">Kết hợp Skills</a>. Kết hợp skills cho phép bạn xây dự án hoàn chỉnh. Ví dụ, kết hợp <code>saas-mvp-launcher</code> + <code>stripe-integration</code> + <code>clerk-auth</code> để xây sản phẩm SaaS đầy đủ với thanh toán và xác thực.
</details>

<details>
<summary><strong>Repo này khác gì các awesome list khác?</strong></summary>

Hầu hết awesome list chỉ liệt kê link đến công cụ. Repo này là danh mục skills thực hành: mỗi skill có mô tả tiếng Việt dễ hiểu, mức độ khó, và — với các skills hàng đầu — prompt mẫu, tổ hợp khuyên dùng, và hướng dẫn khi nào nên dùng lựa chọn thay thế. Tính năng Kết hợp Skills cung cấp công thức dự án từng bước mà bạn không tìm thấy ở đâu khác.
</details>
